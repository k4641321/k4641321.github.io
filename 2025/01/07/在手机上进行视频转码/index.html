在手机上进行视频转码

2025/1/7 转码 FFmpeg 519 字 1 分钟

前言

虽说现在市面上有很多优秀的转换工具了，例如格式工厂，lmo-transform之类的，在github上数不胜数
但是！
却没有一款手机上好用的格式转换工具，大多数都要会员，还有犯人的广告，那么有没有干净好用的格式转换软件呢？

有的！兄弟，有的！

正文

FFmpeg一款优秀的跨平台处理音视频文件的工具包，估计都有所耳闻。

重要的是跨 平 台，既然都跨平台了，那安卓肯定也是可以运行的吧。

准备工作

  • 一个安卓手机
  • termux-app
  • MT管理器(可选)
  • 一个不怎么笨的脑子

首先将上面提到的必要软件下载下来
完成后，打开termux

Termux是一个适用于 Android 的终端模拟器，其环境类似于 Linux 环境。 无需Root或设置即可使用。 Termux 会自动进行最小安装 - 使用 APT 包管理器即可获得其他软件包。

由于软件源默认是国外的，为了下载快一点我们需要换源，这里以清华源为例子

  1. 在终端中输入termux-change-repo来使用带有GUI界面跟换软件源，依次为输入termux-change-repo，回车，选择选项里带有Chinese Mainland字样的选项，等待完成
  2. 用文本编辑器直接编辑$PREFIX/etc/apt/sources.list文件，在原本的源前面加上#号，换行写入deb https://mirrors.tuna.tsinghua.edu.cn/termux/apt/termux-main stable main，保存，使用命令apt update来更新软件源
  3. 使用
sed -i 's@^\(deb.*stable main\)$@#\1\ndeb https://mirrors.tuna.tsinghua.edu.cn/termux/apt/termux-main stable main@' $PREFIX/etc/apt/sources.list

命令更换软件源，同样需要

apt update

接着，我们使用命令

pkg install ffmpeg

来安装ffmpeg，安装完成后，将你要转换的文件放入termux的储存空间里，使用ffmpeg -i 原文件路径 输出文件路径来进行格式转换，等到输出完成，文件就转换好了

没错，就这么简单，结束
